## Deployment


### Podman Compose

```yaml
services:
  authelia-server:
    image: ghcr.io/daemonless/authelia-server:latest
    container_name: authelia-server
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/authelia-server:/config"
    ports:
      - 9091:9091
    restart: unless-stopped
```
